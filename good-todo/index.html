Текущие Выполненные Удалённые

    У вас пока нет ни одной задачи

    1. Начните писать задачу в поле ввода внизу, затем нажмите клавишу 'Enter' или кнопку 'Плюсик' в левом нижем углу, чтобы добавить задачу.

    2. Чтобы отредактировать созданную задачу, просто кликните по ней. Затем в конце просто нажмите "Enter".

    3. Чтобы воспользоваться поиском по задачам, в поле ввода наберите символ '/', затем начните набирать текст. Поиск будет осуществлятся автоматически. Чтобы выйти из режима поиска, очистите поле ввода, нажав кнопку 'Крестик' в левом нижнем углу.

    4. Чтобы удалить все задачи, нажмите на круглую иконку в правом верхнем углу.
